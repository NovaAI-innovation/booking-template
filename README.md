# Adult Creator Website Template

A modern, elegant, and professional multi-page website template for content creators and companions.

## üìÅ How to Use This Template

This project has been templatized with placeholders. Search for and replace the following strings across the codebase:

- `[MODEL_NAME]`: The name of the model/creator.
- `[MODEL_TAGLINE]`: A short tagline (e.g., "Content Creator & Artist").
- `[WELCOME_MESSAGE]`: The hero section welcome text.
- `[SHOP_URL]`: Link to the Shopify or merch store.
- `[BASE_CITY]`: The home base city.

### Services & Pricing (services.html)
- `[SERVICE_1_NAME]`, `[PRICE_1]`, `[SERVICE_1_DESC]`
- `[SERVICE_2_NAME]`, `[PRICE_2]`, `[SERVICE_2_DESC]`
- `[SERVICE_3_NAME]`, `[PRICE_3]`, `[SERVICE_3_DESC]`
- `[FMTY_MINIMUM]`, `[FMTY_PRICE]`

### Platforms (platforms.html)
- `[PLATFORM_1_NAME]`, `[PLATFORM_1_DESC]`, `[PLATFORM_1_URL]`
- `[PLATFORM_2_NAME]`, `[PLATFORM_2_DESC]`, `[PLATFORM_2_URL]`
- `[PLATFORM_3_NAME]`, `[PLATFORM_3_DESC]`, `[PLATFORM_3_URL]`

### Etiquette (etiquette.html)
- `[ETIQUETTE_GUIDELINE_1]`
- `[ETIQUETTE_GUIDELINE_2]`

### Integrations (script.js)
- `[YOUR_EMAILJS_PUBLIC_KEY]`
- `[YOUR_EMAILJS_SERVICE_ID]`
- `[YOUR_EMAILJS_TEMPLATE_ID]`
- `[YOUR_EMAIL_ADDRESS]`
